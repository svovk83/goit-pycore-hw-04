Опис завдань

Task 1 (Salary Analysis): Функція для аналізу фінансових звітів. Розраховує загальну та середню зарплату, ігноруючи пошкоджені дані.

Task 2 (Cats Database): Парсинг неструктурованих текстових даних у структурований список словників для подальшої обробки в AI-моделях.

Task 3 (Directory Visualizer): CLI-утиліта для візуалізації структури папок. Використовує colorama для диференціації файлів та директорій, що зменшує когнітивне навантаження на користувача.

Task 4 (Assistant Bot): Прототип CLI-інтерфейсу для управління контактами. Реалізовано парсинг команд та базову бізнес-логіку (Add, Change, Show All).
